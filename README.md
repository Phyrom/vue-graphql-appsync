## Getting started    

1. clone project    

```
git clone https://github.com/dabit3/vue-graphql-appsync.git
```

2. cd into directory    

```
cd vue-graphql-appsync
```

3. install dependencies using npm or yarn    

```
yarn || npm install
```

4. update your credentials in `src/AppSync.js`
